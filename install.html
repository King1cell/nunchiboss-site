macOS용 다운로드 .dmg · Apple Silicon 전용 · macOS 14+
  1. 1받은 .dmg를 열고, 눈치상사를 응용 프로그램 폴더로 드래그하세요.
  2. 2앱을 처음 실행하면 아래 안내창이 뜹니다. 설정 열기를 누르세요. 키보드·마우스의 횟수만 집계하며, 무엇을 눌렀는지는 기록하지 않습니다.
  3. 3시스템 설정의 손쉬운 사용이 열립니다. 목록에 보이는 BossMascot이 눈치상사입니다(앱의 내부 이름). 그 토글을 켜세요.
  4. 4토글을 켜면 Mac 암호를 물어봅니다. 암호를 입력하고 설정 수정을 누르세요.
  5. 5토글이 파랗게 켜지면 권한 설정이 끝났습니다.
  6. 6완료! 상사가 화면 구석에 나타납니다. 트레이 아이콘 → 설정…에서 캐릭터·이름·크기·위치를 바꿀 수 있어요. 화나서 두드릴수록 굽신거립니다.
혹시 안 열리나요?
이 앱은 Apple 서명·공증을 받아 별도 우회 없이 열립니다. 그래도 막히면 앱을 Control-클릭 → 열기. Intel Mac은 지원하지 않습니다(Apple Silicon 전용).
Windows용 다운로드 .exe · Windows 10/11 · 64-bit
⚠️ "Windows의 PC 보호" 파란 창이 뜨면 (정상입니다)
아직 서명 전이라 뜨는 경고입니다. 바이러스가 아닙니다 — 창에서 추가 정보를 누른 뒤 실행을 누르면 됩니다.
  1. 1다운로드한 .exe를 실행하면 “Windows의 PC 보호” 파란 창이 뜹니다. 왼쪽의 추가 정보를 누르세요.
  2. 2실행 버튼이 나타납니다. 눌러서 설치를 시작하세요. 게시자가 ‘알 수 없음’인 건 아직 서명 전이라 그럴 뿐, 바이러스가 아닙니다.
  3. 3설치가 진행됩니다. 잠시만 기다리세요.
  4. 4설치가 끝나면 작업 표시줄 오른쪽 트레이에 상사 아이콘이 나타납니다. 안 보이면 ∧를 눌러 숨은 아이콘을 펼치세요.
  5. 5완료! 트레이 아이콘을 우클릭 → 설정…에서 캐릭터·이름·크기·위치를 바꿀 수 있어요. 화나서 두드릴수록 굽신거립니다.
왜 경고가 뜨나요?
개별 개발자용 Windows 코드서명은 비용·심사 문제로 아직 준비 중입니다. 다운로드가 늘면 경고는 점차 사라지며, 소스 공개 후 무료 서명(SignPath) 도입도 검토 중입니다. 앱은 여전히 키 입력 내용을 기록하거나 외부로 전송하지 않습니다.
