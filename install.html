macOS용 다운로드 .dmg · Apple Silicon 전용 · macOS 14+
1받은 .dmg를 열고, 눈치상사를 응용 프로그램 폴더로 드래그하세요.
2처음 실행하면 시스템 설정 › 개인정보 보호 및 보안 › 손쉬운 사용에서 눈치상사를 켜주세요. 목록에는 내부 이름 BossMascot으로 표시됩니다. 키보드·마우스의 횟수만 감지하며, 무엇을 눌렀는지는 기록하지 않습니다.
3끝났습니다. 상사가 화면 구석에 나타납니다. 화나서 두드릴수록 굽신거립니다.
혹시 안 열리나요?
이 앱은 Apple 서명·공증을 받아 별도 우회 없이 열립니다. 그래도 막히면 앱을 Control-클릭 → 열기. Intel Mac은 지원하지 않습니다(Apple Silicon 전용).
Windows용 다운로드 .exe · Windows 10/11 · 64-bit
⚠️ "Windows의 PC 보호" 파란 창이 뜨면 (정상입니다)
아직 서명 전이라 뜨는 경고입니다. 바이러스가 아닙니다 — 창에서 추가 정보를 누른 뒤 실행을 누르면 됩니다.
1받은 설치 파일을 실행합니다.
2파란 경고 창이 뜨면 추가 정보 → 실행 (위 안내 참고).
3설치가 끝나면 상사가 화면 구석에 나타납니다.
왜 경고가 뜨나요?
개별 개발자용 Windows 코드서명은 비용·심사 문제로 아직 준비 중입니다. 다운로드가 늘면 경고는 점차 사라지며, 소스 공개 후 무료 서명(SignPath) 도입도 검토 중입니다. 앱은 여전히 키 입력 내용을 기록하거나 외부로 전송하지 않습니다.
